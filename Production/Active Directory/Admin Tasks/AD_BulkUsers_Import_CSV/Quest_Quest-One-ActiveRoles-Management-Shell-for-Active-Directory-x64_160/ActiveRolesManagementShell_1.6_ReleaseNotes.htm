Welcome to ActiveRoles Management Shell

Quest® One ActiveRoles® Management Shell for Active Directory is an Active Directory-specific automation and scripting shell that provides a command-line management interface for administering directory data either via Quest One ActiveRoles or by directly accessing Active Directory domain controllers. ActiveRoles Management Shell is built on Microsoft Windows PowerShell technology.

Upgrade and Compatibility

To upgrade an earlier version of ActiveRoles Management Shell, run the Setup program for installing ActiveRoles Management Shell 1.6. The Setup program removes the earlier version of Management Shell and install version 1.6.

Version 1.6 of ActiveRoles Management Shell is compatible with Quest One ActiveRoles of version 6.8 only. This has the following implications:

  • Version 1.6 of ActiveRoles Management Shell cannot be installed on a computer that hosts ActiveRoles components of version other than 6.8. For example, if version 6.7 of the Administration Service, MMC Interface or Web Interface is installed on a computer, you receive an error when attempting to install version 1.6 of ActiveRoles Management Shell on that computer. The error message informs you of a conflict with the software that exists on the computer, and advises you to install ActiveRoles Management Shell on a different computer.
  • The ActiveRoles Management Shell cmdlets of version 1.6 cannot connect to the ActiveRoles Administration Service of version other than 6.8. For example, if you use the Proxy connection parameter in conjunction with the Service parameter that specifies a computer running the ActiveRoles Administration Service of version 6.7, you receive an error. The error message informs you that the connection cannot be established because of version incompatibility.

When upgrading Quest One ActiveRoles to version 6.8, be aware that version 6.8 of the ActiveRoles Administration Service requires ActiveRoles Management Shell of version 1.6. The Administration Service Setup program installs version 1.6 of ActiveRoles Management Shell during the upgrade process so that the resulting installation meets the version compatibility requirements.

System Requirements

Before installing ActiveRoles Management Shell, ensure your system meets the following minimum hardware and software requirements:

Platform 1 GHz or higher Intel Pentium-compatible CPU.
Memory 1 GB or more recommended.
Hard Disk Space About 100 MB of free disk space.
Operating System
  • Microsoft Windows Server 2003, 32-bit (x86) or 64-bit (x64), Service Pack 2 or later
  • Microsoft Windows Server 2003 R2, 32-bit (x86) or 64-bit (x64), with or without any Service Pack
  • Microsoft Windows XP, 32-bit (x86) or 64-bit (x64), Service Pack 3
  • Microsoft Windows Server 2008, Standard or Enterprise edition, 32-bit (x86) or 64-bit (x64), Service Pack 2 or later
  • Microsoft Windows Vista, Ultimate, Business or Enterprise edition, 32-bit (x86) or 64-bit (x64), Service Pack 2 or later
  • Microsoft Windows Server 2008 R2, Standard or Enterprise edition, with or without any Service Pack
  • Microsoft Windows 7, Ultimate, Professional or Enterprise edition, 32-bit (x86) or 64-bit (x64), with or without any Service Pack
  • Microsoft Windows Server 2012, Standard or Datacenter edition, with or without any Service Pack
  • Microsoft Windows 8, Professional or Enterprise edition, 32-bit (x86) or 64-bit (x64), with or without any Service Pack
Microsoft .NET Framework Microsoft .NET Framework 3.5 SP1.
Microsoft Windows PowerShell Windows PowerShell 2.0 or later.
